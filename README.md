# Прогнозирование удовлетворённости и увольнения сотрудников компании «Работа с заботой»

Две ML-модели для HR-аналитики: предсказание уровня удовлетворённости работой и вероятности увольнения сотрудника.

## Данные
- **Источник:** HR-департамент компании «Работа с заботой»
- **Размер:** 4000 сотрудников (обучающая выборка), 2000 (тестовая)
- **Признаки:** департамент, уровень должности, загруженность, стаж, повышения, нарушения, оценка руководителя, зарплата

## Ход работы
- Предобработка: устранение пропусков (мода), исправление орфографических ошибок, проверка дубликатов
- EDA: анализ распределений признаков в тренировочной и тестовой выборках
- Корреляционный анализ (Phik) для обеих выборок
- **Задача 1 (регрессия):** предсказание уровня удовлетворённости (job_satisfaction_rate) через Pipeline + RandomizedSearchCV
- **Задача 2 (классификация):** предсказание факта увольнения (quit) с добавлением предсказанного уровня удовлетворённости как нового признака
- Анализ портрета уволившегося сотрудника
- Бизнес-рекомендации для HR-департамента

## Ключевые выводы
- **Модель удовлетворённости:** DecisionTreeRegressor, SMAPE = 14.49 (порог ≤ 15)
- **Модель увольнения:** SVC, ROC-AUC = 0.93 (порог ≥ 0.91)
- **Ключевые факторы риска увольнения:** низкая зарплата (средняя у уволившихся — 23 885 vs 37 702 у оставшихся), стаж до 3 лет, отсутствие повышений
- **Уровень удовлетворённости** использован как дополнительный признак во второй задаче — связь с увольнением подтверждена корреляцией (0.61)
- **Рекомендации:** пересмотр политики оплаты труда, программы адаптации для новых сотрудников, повышение прозрачности карьерного роста
