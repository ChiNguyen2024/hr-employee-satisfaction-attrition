# Прогнозирование снижения покупательской активности клиентов интернет-магазина «В один клик»

ML-модель для предсказания оттока клиентов и приоритизации удержания на основе поведенческих и финансовых данных.

## Данные
- **Источник:** внутренние данные интернет-магазина «В один клик»
- **Размер:** 1300 клиентов, 4 исходных датасета
- **Признаки:** маркетинговые коммуникации, поведение на сайте, покупательская активность, выручка по периодам, прибыльность клиента

## Ход работы
- Предобработка данных: устранение явных и неявных дубликатов, проверка на пропуски
- Формирование аналитического датасета: объединение 4 таблиц, декомпозиция временных признаков (выручка и время на сайте по периодам)
- EDA: анализ распределений, проверка на нормальность (тест Шапиро-Уилка)
- Корреляционный анализ (Phik) и проверка мультиколлинеарности (VIF)
- Построение Pipeline с ColumnTransformer: OHE, OrdinalEncoder, StandardScaler/MinMaxScaler/RobustScaler
- Подбор лучшей модели через GridSearchCV, RandomizedSearchCV и OptunaSearchCV
- SHAP-анализ важности признаков
- Сегментация клиентов по прибыльности и приоритету удержания

## Ключевые выводы
- **Лучшая модель:** логистическая регрессия, ROC AUC = 0.895
- **Ключевые предикторы оттока:** количество страниц за визит, время на сайте, среднее число просмотренных категорий — высокие значения снижают риск оттока; акционные покупки и неоплаченные товары в корзине — повышают
- **Малоинформативные признаки:** популярная категория, тип сервиса, разрешение на рассылку
- **Сегментация:** клиенты разделены на три группы по приоритету удержания (высокий / средний / низкий) на основе прибыльности и предсказанного уровня активности
- **Бизнес-рекомендация:** лояльным клиентам — программы лояльности и персональные подборки; клиентам из группы риска — email-напоминания об отложенных товарах и персональные акции
